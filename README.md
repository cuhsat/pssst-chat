Pssst Chat
==========
A simple `ncurses` based chat for [Pssst](https://github.com/cuhsat/pssst).

![Pssst Chat](doc/pssst-chat.png)

Usage
-----
```
$ pssst-chat [option|username|~]
```

Options
-------
* `-h` Shows the usage
* `-l` Shows the license
* `-v` Shows thw version
